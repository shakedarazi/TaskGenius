# Sequence Flows

## 1. Task Creation via Chatbot

```
┌────────┐     ┌──────────┐     ┌─────────────────┐     ┌─────────┐
│ Client │     │ core-api │     │ chatbot-service │     │ MongoDB │
└───┬────┘     └────┬─────┘     └────────┬────────┘     └────┬────┘
    │               │                    │                   │
    │ POST /chat    │                    │                   │
    │──────────────▶│                    │                   │
    │               │                    │                   │
    │               │ POST /interpret    │                   │
    │               │───────────────────▶│                   │
    │               │                    │                   │
    │               │   {intent, entities}                   │
    │               │◀───────────────────│                   │
    │               │                    │                   │
    │               │ Insert task        │                   │
    │               │───────────────────────────────────────▶│
    │               │                    │                   │
    │               │ Confirmation       │                   │
    │               │◀───────────────────────────────────────│
    │               │                    │                   │
    │  Response     │                    │                   │
    │◀──────────────│                    │                   │
    │               │                    │                   │
```

## 2. Direct Task Creation

```
┌────────┐     ┌──────────┐     ┌─────────┐
│ Client │     │ core-api │     │ MongoDB │
└───┬────┘     └────┬─────┘     └────┬────┘
    │               │                │
    │ POST /tasks   │                │
    │──────────────▶│                │
    │               │                │
    │               │ Insert         │
    │               │───────────────▶│
    │               │                │
    │               │ Result         │
    │               │◀───────────────│
    │               │                │
    │  201 Created  │                │
    │◀──────────────│                │
    │               │                │
```

## 3. Weekly Summary Generation

```
┌───────────┐     ┌──────────┐     ┌─────────┐
│ Scheduler │     │ core-api │     │ MongoDB │
└─────┬─────┘     └────┬─────┘     └────┬────┘
      │                │                │
      │ Trigger        │                │
      │───────────────▶│                │
      │                │                │
      │                │ Query tasks    │
      │                │───────────────▶│
      │                │                │
      │                │ Task data      │
      │                │◀───────────────│
      │                │                │
      │                │ Generate summary (AI)
      │                │────────┐       │
      │                │        │       │
      │                │◀───────┘       │
      │                │                │
      │                │ Store summary  │
      │                │───────────────▶│
      │                │                │
```

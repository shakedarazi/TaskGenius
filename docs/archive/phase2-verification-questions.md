# שאלות לבדיקה - Phase 0, 1, 2

## 🎯 מטרה
לוודא שכל מה שעשינו עד עכשיו עובד טוב לפני שנעבור ל-Phase 3.

---

## ✅ Phase 0: Infrastructure & Testing Setup

### בדיקות בסיסיות:
- [ ] הבוט עונה על כל הודעה (לא קורס)
- [ ] התשובות תמיד כוללות `reply`, `intent`, `suggestions`
- [ ] אין שגיאות בקונסול/לוגים

---

## ✅ Phase 1: LLM Integration

### בדיקת LLM (אם יש API key תקין):
- [ ] "What are my tasks?" → תשובה מ-LLM (אם יש quota)
- [ ] "מה המשימות שלי?" → תשובה מ-LLM בעברית (אם יש quota)

### בדיקת Fallback (אם אין quota):
- [ ] "What are my tasks?" → תשובה מ-rule-based (אם אין quota)
- [ ] "מה המשימות שלי?" → תשובה מ-rule-based בעברית (אם אין quota)
- [ ] אם יש בעיית quota → מופיעה הודעה: "⚠️ הגעת למגבלת ה-quota..."

## ✅ Phase 2: Intent Quality & Task-Aware Reasoning

### 1. Task Insights (תובנות על משימות)

#### בעברית:
- [ ] "מה דחוף לי?"
- [ ] "מה המשימות בעדיפות גבוהה?"
- [ ] "מה יש לי עם תאריך יעד?"
- [ ] "תראה לי מה דחוף"
- [ ] "איזה משימות דחופות?"

#### באנגלית:
- [ ] "What's urgent for me?"
- [ ] "Show me high priority tasks"
- [ ] "What tasks are due soon?"
- [ ] "What's coming up?"
- [ ] "Which tasks need attention?"

**תוצאה צפויה:**
- Intent: `task_insights`
- תשובה מתייחסת למשימות בפועל
- מזהה משימות דחופות/בעדיפות גבוהה

---

### 2. Potential Create (יצירת משימה - דורש הבהרה)

#### בעברית:
- [ ] "תוסיף משימה"
- [ ] "צור משימה חדשה"
- [ ] "הוסף משימה"
- [ ] "תוסיף לי משימה"

#### באנגלית:
- [ ] "add task"
- [ ] "create a new task"
- [ ] "add a task"
- [ ] "I want to create a task"

**תוצאה צפויה:**
- Intent: `potential_create`
- שואל על כותרת המשימה
- מציע אפשרויות (תאריך יעד, עדיפות, וכו')

---

### 3. Potential Update (עדכון משימה - דורש הבהרה)

#### בעברית:
- [ ] "עדכן משימה"
- [ ] "שנה משימה"
- [ ] "ערוך משימה"
- [ ] "תעדכן את המשימה הראשונה"

#### באנגלית:
- [ ] "update task"
- [ ] "change a task"
- [ ] "modify task"
- [ ] "edit the first task"

**תוצאה צפויה:**
- Intent: `potential_update`
- שואל איזו משימה לעדכן (אם יש כמה)
- מציג רשימת משימות רלוונטיות
- שואל מה לשנות

---

### 4. Potential Delete (מחיקת משימה - דורש אישור)

#### בעברית:
- [ ] "מחק משימה"
- [ ] "הסר משימה"
- [ ] "תמחק את המשימה הראשונה"
- [ ] "תמחק משימה"

#### באנגלית:
- [ ] "delete task"
- [ ] "remove a task"
- [ ] "delete the first task"
- [ ] "remove task"

**תוצאה צפויה:**
- Intent: `potential_delete`
- שואל איזו משימה למחוק (אם יש כמה)
- מזהיר על מחיקה לצמיתות
- מבקש אישור

---

### 5. List Tasks (רשימת משימות)

#### בעברית:
- [ ] "מה המשימות שלי?"
- [ ] "תראה לי את המשימות"
- [ ] "רשימת משימות"
- [ ] "הצג משימות"

#### באנגלית:
- [ ] "list my tasks"
- [ ] "show me tasks"
- [ ] "what are my tasks?"
- [ ] "show tasks"

**תוצאה צפויה:**
- Intent: `list_tasks`
- מציג את המשימות
- תשובה מגוונת (לא אותה תשובה כל פעם)

---

### 6. Weekly Summary (סיכום שבועי)

#### בעברית:
- [ ] "תראה לי סיכום שבועי"
- [ ] "מה הסיכום השבועי?"
- [ ] "סיכום"
- [ ] "דוח שבועי"

#### באנגלית:
- [ ] "show me weekly summary"
- [ ] "what's my weekly summary?"
- [ ] "summary"
- [ ] "weekly report"

**תוצאה צפויה:**
- Intent: `get_insights`
- מציג סיכום שבועי (אם יש)
- או מבקש לאסוף נתונים

### בדיקות (אם יש בעיית quota):

- [ ] ההודעה בעברית אם שולחים בעברית
